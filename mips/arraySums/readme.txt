Program performs the following:

       num
total = ∑ factor+(a[i] * b[i])
        1


User can do the following:
- Run program and check the result – the value of total.
- Change the lengths of the arrays a and b and the value of num, then rerun the program.
- Change the value of init and rerun the program.
- Change the value init back to 2 and change the value of factor and rerun the program.
