Running the Program:
1) Create new folder, any name is fine.
2) Place the webserver.c program into your recently made folder.
3) Place the browser files you want to access into the same folder as the executable.
4) Open up a command line application, like Terminal, and open up the file path of the folder you recently made.
5) To compile, type: gcc -o webserve webserver.c
6) Files are ready for browsing, enjoy.
