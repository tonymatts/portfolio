User's Guide V2.1Database for a Hypothetical Recording Studio By Tony MattsVisit: http://web-dev.cs.umt.edu/~am154932/studioDbInterface.html ~ link no longer active.Below are a list of eleven functions, some affecting multiples tables, along with actual data from the database. The available functions allow the user to:- Insert an album.- Insert an engineer.- Delete a record company.- Delete a producer.- Update album's record company (update rows 2 - 10).- Update software version.- List All Musicians and their Producer.- List All Albums and Record Company Info.- List Studio Techs and the Albums they've worked on.- List Producers and the Groups they've produced.- List all Guitars made by (Gibson knock-off brand) "Gibbons".File - insertAlbum.phpMASTER_COPY_A AlbumCode Title RcName (1,'Beastmaster','Nates'), (2,'Choppin','Nates'), (3,'Surfing','Laceys'), (4,'Roku','Nates'), (5,'Squareone','Bens'), (6,'Spaceship','Bens'), (7,'Orange','Nates'), (8,'Banned','Laceys'), (9,'Founded','Bens'), (10,'Knocking','Laceys');Insert Functions
